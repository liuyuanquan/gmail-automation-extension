# 更新日志

## [1.0.3] - 2025-11-30

### 新增功能
- 优化 Excel 导出样式，支持列宽、行高、超链接等格式
- Email 列自动添加超链接，点击可直接发送邮件
- 所有单元格内容垂直居中显示

### 改进
- 优化 Excel 文件样式设计，提升美观度
- 表头使用深蓝灰色背景和白色加粗文字
- 状态列使用颜色区分（已发送=绿色，发送失败=红色）
- 优化列宽设置，根据内容自动调整
- 优化 manifest.json 配置，移除不必要的权限
- 优化 vite.config.js，使用现代 ES 模块语法

### 修复
- 修复图标配置问题，使用 PNG 格式替代 SVG
- 修复 Service Worker 注册错误

### 清理
- 移除通知功能相关代码（Chrome 通知权限问题）
- 删除 postcss.config.js（Vite 已内置 PostCSS 支持）
